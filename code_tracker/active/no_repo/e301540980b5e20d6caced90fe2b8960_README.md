This is the frontend for the Scout DB application.

## Getting Started

You can run the application from the project root (`C:\SCOUTNEW`) or from this directory.

### From Root (`C:\SCOUTNEW`)

```bash
npm install   # Installs dependencies for the frontend
npm run dev   # Starts the development server
```

### From Here (`C:\SCOUTNEW\scout_db\frontend`)

```bash
npm install
npm run dev
```

## Scripts

- `dev`: Start the development server
- `build`: Build for production
- `preview`: Preview the production build
- `lint`: Lint the code
